ABOUT THE GAME
------------------------------------
Data Drift is a fast-paced data survival game where the player avoids
incoming data anomalies while adapting to increasing system instability.

The game blends motion, visual feedback, and adaptive difficulty to
simulate a "data corruption survival" environment.

------------------------------------
FEATURES (v1.0.0)
------------------------------------
- Phase-based difficulty system
- Dynamic enemy spawn scaling
- Screen shake & juice effects
- Trail-based player motion
- Collision-based survival scoring
- Fullscreen support (F11 toggle)
- Offline playable executable

------------------------------------
CONTROLS
------------------------------------
Left Arrow  - Move Left
Right Arrow - Move Right
R           - Restart (after game over)
F11         - Toggle Fullscreen
