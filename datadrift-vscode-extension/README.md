# Data Drift - Marketplace Listing

**Data Drift** is a lightweight VS Code extension that launches a fast, distraction-free break game directly inside your coding workflow. It helps developers reset focus, reduce mental fatigue, and improve productivity through short interactive break sessions.

---

## What it does

Data Drift integrates a mini game directly into VS Code so you can take quick focus breaks without leaving your editor.

With one click, you can launch the game and step away from coding for a short mental reset.

---

## ✨ Features

* Instant game launch from VS Code sidebar
* Designed for quick focus and break sessions
* Helps reduce coding fatigue and improve concentration
* One-click play from activity bar
* Fully offline (no internet required)
* Bundled executable (no setup or dependencies)

---

## How to use

1. Install the extension
2. Open VS Code sidebar
3. Click **Data Drift**
4. Press **Play Data Drift**
5. Take a short focus break and return refreshed

---

## About the game

Data Drift is a lightweight reflex-based mini game designed for developers who work long hours. It is optimized for:

* Pomodoro breaks
* Focus resets
* Short mental recovery sessions during coding

---

## Requirements

* Visual Studio Code ^1.85.0 or later
* No additional installation required
